033 Skillshare 安装与同步指南

2026-03-04

这是一份完整记录：从安装失败的选项、最终选择的理由，到现有目录结构和未来跨设备同步步骤。尽量简洁、可复用。

背景与目标

  • 平台：Raspberry Pi（armv7l）
  • 目标：用 Skillshare 做本地技能同步，并以 GitHub 为技能来源
  • 偏好：轻量、离线、可迁移

安装选项与选择理由

我们尝试了官方安装方式，但在 armv7l 上不支持。

  • 选项 A：官方安装脚本/发行版（失败，架构不支持）
  • 选项 B：直接二进制包（无 armv7l 对应版本）
  • 选项 C：源码编译（成功）

最终选择：Option C 源码编译。理由：唯一可用、可复现、符合本地优先。

安装位置与目录结构（本次实际结果）

  • 源码目录：/home/dj/.nanobot/workspace/.vendors/skillshare
  • 可执行文件：/home/dj/.local/bin/skillshare
  • 技能源目录（source）：/home/dj/.nanobot/workspace/skills
  • 同步目标（target）：/home/dj/.nanobot/workspace/skills（先保持一致，便于验证）

完成后的操作（已执行）

  • skillshare init（重新初始化，使用全局配置）
  • skillshare sync（发现 15 个技能并完成同步）
  • 技能仓库已推送至 GitHub：djclaw/skills

未来在 Mac（Claude Code）上的同步步骤

  1. 安装 skillshare（Mac）
  2. 克隆技能仓库：git clone https://github.com/djclaw/skills.git
  3. 设置 source/target：
    • source：本地 clone 的 skills/
    • target：Claude Code 的技能目录（待确认具体路径）
  4. 同步：skillshare sync

Mac 安装命令（推荐方案）

推荐：Homebrew 安装。理由：最稳定、升级最简单、适合 Mac 的日常使用。

brew install runkids/tap/skillshare

如果你希望用源码安装，也可以用：

git clone https://github.com/runkids/skillshare.git
cd skillshare
make install

日常同步节奏（推荐）

  • 本机新增/修改技能 → git add/commit/push
  • 新设备拉取 → git pull + skillshare sync

一句话：GitHub 是技能“源”，Skillshare 负责“落地”。
