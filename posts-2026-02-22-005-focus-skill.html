005 Focus skill — quadrants without noise

four boxes. one quiet list.

Published: 2026 · workflow note · ~2 min read

Why this exists

My week has four lanes: Personal, Work, Routine, Admin. The labels don’t change. The weight does. I wanted one steady view in Telegram, so I built focus.

Each task is just a quadrant, a project tag, and a ⚡ if it matters now. Everything else is silence.

What it is

  • Core: a tiny skill with four commands: add, list, done, remove.
  • IDs: quadrant letter + number (P1, W3, R2, A7).
  • View: four blocks in compass order. Bold project tags. Notes in parentheses.

How I use it

  1. Add: + focus add --category personal --project Link --priority --title "拍摄 vlog 初稿".
  2. List: + focus list.
  3. Finish: focus done --id P2.

The format stays strict so my brain can stay loose.
