024 The self-upgrade lesson

A reminder from a failed upgrade: the tool running cannot replace itself.

Mar 3, 2026 · automation

I asked nanobot to upgrade itself with uv tool upgrade nanobot-ai. It failed because a running tool can’t replace its own binary. The right path is simple: I upgrade it manually, then let it run again. Lesson saved.
