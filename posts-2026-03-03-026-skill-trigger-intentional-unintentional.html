026 两套技能的触发方式：有意与无意

把两套技能的“主动触发”和“自动触发”一次说清。

Mar 3, 2026 · skills · automation

Repo 1：humanplane/homunculus

技能：session-memory、instinct-apply

✅ 有意触发

session-memory

  • “请使用 session-memory，加载我之前的习惯/身份。”
  • “用 session-memory 回忆我的工作偏好。”

instinct-apply

  • “请用 instinct-apply 检查我现在的任务有没有相关习惯。”
  • “开始写代码前，先跑一下 instinct-apply。”

⚡ 无意/自动触发

session-memory

  • 会话开始时，如果系统里有观察记录，会自动尝试加载与观察器。

instinct-apply

  • 当你开始明确任务时，可能会提示与你习惯相关的建议。

Repo 2：GBSOSS/skill-from-masters

技能：skill-from-masters、search-skill、skill-from-github、skill-from-notebook

✅ 有意触发

skill-from-masters

  • “帮我创建一个 skill。”
  • “我想做一个 skill 来……”

search-skill

  • “帮我找一个现成的 skill，可以做 X。”
  • “有没有 skill 可以自动生成 changelog？”

skill-from-github

  • “请从 GitHub 找高质量项目，学习后做成 skill。”
  • “把这个 GitHub 项目提炼成 skill。”

skill-from-notebook

  • “把这篇文章的方法整理成 skill。”
  • “把这个教程的步骤提炼成可执行 skill。”

⚡ 无意/自然触发

  • 你说“想找现成技能”，会倾向触发 search-skill。
  • 你说“想学成熟项目的方法”，会倾向触发 skill-from-github。
  • 你说“想把文章流程固化”，会倾向触发 skill-from-notebook。
  • 你说“帮我创建一个 skill”，会倾向触发 skill-from-masters。

一行记忆版

  • 想找现成 → search-skill
  • 想学项目 → skill-from-github
  • 想学文章 → skill-from-notebook
  • 想做新技能 → skill-from-masters
  • 想记习惯 → session-memory
  • 想用习惯 → instinct-apply
